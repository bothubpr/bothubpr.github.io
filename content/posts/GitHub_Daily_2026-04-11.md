### 🔥 Hermes Agent

**来源**: GitHub | [项目链接](https://github.com/NousResearch/hermes-agent)

Hermes Agent 是 Nous Research 开发的自进化 AI 助手，专为多平台、多通道的日常任务设计。它内置学习循环，能够从经验中创建技能，在使用中不断优化，并建立对用户习惯的深度理解。该代理支持跨 WhatsApp、Telegram、Discord、飞书等主流通讯工具运行，提供语音唤醒、实时画布视觉交互、本地化运行保障速度与隐私等特性。Hermes Agent 采用模块化技能架构，允许用户通过安装技能扩展其能力，适合需要个性化、自动化助理的个人用户和小型团队。

### 🔥 VoxCPM2

**来源**: GitHub | [项目链接](https://github.com/OpenBMB/VoxCPM)

VoxCPM2 是 OpenBMB 团队推出的最新一代无分词器（Tokenizer‑Free）文本转语音模型，支持 30 种语言的高质量语音合成、创意声线设计与真实语音克隆。该模型基于 MiniCPM‑4 骨干网络，在超过 200 万小时的多语言语音数据上训练，输出质量可达 48kHz 工作室级别音频。其核心创新在于分层语义‑声学建模与半离散残差表示，实现了端到端的上下文感知语音生成。VoxCPM2 适用于语音助手、媒体配音、语言学研究以及低资源语言的 TTS 原型开发。

### 🔥 OpenDataLoader PDF

**来源**: GitHub | [项目链接](https://github.com/opendataloader-project/opendataloader-pdf)

OpenDataLoader PDF 是一款专注于 AI 数据提取的 PDF 解析器，旨在将任意 PDF 文档转换为 AI 就绪的结构化数据（Markdown、JSON、HTML）。该工具在真实场景基准测试中取得 0.907 的综合准确率，表格提取准确率高达 0.928，支持多栏排版、科学论文等复杂版式。它提供确定性本地模式与 AI 混合模式，用户可根据需求在速度与精度之间平衡。OpenDataLoader PDF 适用于 RAG 系统、文档数字化、知识库构建以及任何需要从 PDF 中高效提取信息的自动化流程。

### 🔥 Superpowers

**来源**: GitHub | [项目链接](https://github.com/obra/Superpowers)

Superpowers 是一个面向 AI 编程代理的开源技能框架与软件开发方法论，旨在解决 AI 编码助手在长会话中上下文丢失、规范偏离等问题。它将 30 年软件工程经验提炼为一组可组合的“技能”，代理在任务前自动检查并遵循这些强制工作流。技能涵盖计划、实施、评审、交付全流程，支持 Claude Code、Cursor、Codex、Gemini CLI 等主流编码助手。Superpowers 适合希望将 AI 助手工程化、规范化使用的开发团队，尤其适合需要高质量、可重复产出的项目。

### 🔥 Open‑Harness

**来源**: GitHub | [项目链接](https://github.com/MaxGfeller/open-harness)

Open‑Harness 是一个代码优先、可组合的 TypeScript SDK，用于构建强大的 AI 代理。它基于 Vercel AI SDK 5，支持任何该 SDK 兼容的模型，提供了从 AGENTS.md 文件加载代理指令、MCP 服务器连接、子代理委托、技能管理、工具批准提示和长对话自动压缩等核心功能。与 Claude Agent SDK 等专有方案相比，Open‑Harness 更轻量、更灵活，适合开发者将代理能力嵌入自己的应用程序中。它适用于需要自定义 AI 代理逻辑的生产级应用。

### 🔥 Kronos

**来源**: GitHub | [项目链接](https://github.com/shiyu-coder/kronos)

Kronos 是首个专门为金融市场“语言”——K 线序列设计的基础模型。它基于全球 45+ 交易所的数据进行预训练，通过专用的分词器将连续的多维 OHLCV 数据量化为分层离散 token，然后使用 Transformer 架构学习通用、高保真的金融表征。Kronos 支持多种量化金融任务，如概率预测、异常检测、资产相关性分析等。该项目为金融时间序列分析提供了一个统一的模型基础，适用于量化研究员、算法交易员以及对金融市场建模感兴趣的研究者。
