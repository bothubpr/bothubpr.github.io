### Ntsc-rs：开源模拟电视与 VHS 画面特效库

**来源**: HN
**摘要**: Ntsc-rs 是一款开源的视频模拟库，能够实时模拟老式模拟电视和 VHS 录像带的画面失真效果，包括雪花噪点、色彩偏移、扫描线等经典复古视觉元素。该项目在 GitHub 上获得了极高的关注度，为影视创作和复古游戏爱好者提供了强大的工具。
**链接**: [ntsc.rs](https://ntsc.rs/)

### 第 29 届国际混淆 C 代码大赛 (IOCCC) 2025 获奖作品公布

**来源**: HN
**摘要**: 第 29 届 IOCCC 获奖作品正式揭晓，展示了 C 语言在代码混淆艺术上的极致创造力。这些作品以晦涩难懂却功能正常著称，是对编程技巧的极致挑战，也成为编程文化中独特的艺术表达形式。
**链接**: [IOCCC 2025](https://www.ioccc.org/2025/)

### Valve P2P 网络故障持续超过两个月

**来源**: HN
**摘要**: Valve 的 GameNetworkingSockets P2P 网络服务已出现严重故障超过两个月，严重影响 Steam 联机游戏体验。社区在 GitHub 上持续追踪，积累 228 分和 111 条评论，呼吁官方尽快修复底层协议问题。
**链接**: [GitHub Issue](https://github.com/ValveSoftware/GameNetworkingSockets/issues/398)

### 社区呼吁 Anthropic 推出官方 Linux 版 Claude Desktop

**来源**: HN
**摘要**: 随着 Claude 在开发者社区的广泛使用，Linux 用户强烈呼吁 Anthropic 推出官方支持的 Linux 桌面客户端。目前 Linux 用户只能依赖第三方方案，存在稳定性与安全隐患，官方原生支持将极大提升开发者体验。
