### 🔥 TradingAgents：多智能体 LLM 金融交易框架

**来源**: GitHub | [项目链接](https://github.com/TauricResearch/TradingAgents)

由 TauricResearch 团队开发，基于 LangGraph 构建的多智能体金融交易框架，模拟真实交易公司运作流程——让 LLM 扮演基本面分析师、情绪分析师、技术分析师、交易员和风控经理，协同制定决策。目前 Star 数已突破 63,000，日增超 2,200 星。支持 DeepSeek、Qwen、GLM、Azure 等多种模型提供商，已发布结构化输出代理（研究经理、交易员、投资组合经理）和生产级回测功能。适用于量化交易研究、资产管理和个人投资分析。

### 🔥 Ruflo：Claude 多智能体编排平台

**来源**: GitHub | [项目链接](https://github.com/ruvnet/ruflo)

🌊 一款企业级 AI Agent 编排平台，专为 Claude Code 设计。支持部署智能多 Agent Swarm，协调自主工作流，构建对话式 AI 系统。核心特性包括：64 个跨 16 类别的专用 Agent、蜂群智能协调（Queen-led 架构）、RAG 集成、87 个高级 MCP 工具、自学习/自优化架构。支持网格、层级、环形和星形四种 swarm 拓扑。已从原 Claude Flow 更名并发布 v3.0.0-alpha，深度集成了 ReasoningBank、Router 等子模块。

### 🔥 Maigret：OSINT 用户名侦查工具

**来源**: GitHub | [项目链接](https://github.com/soxoj/maigret)

🕵️‍♂️ 一款强大的开源情报（OSINT）工具，仅通过用户名即可从 3,000+ 站点收集人物的完整数字档案。无需 API 密钥，默认对 500 个热门站点并行搜索。支持 HTML/PDF 报告导出、个人信息提取、递归搜索（自动跟踪发现的 ID 和用户名）、按标签/国家/类别筛选站点、Tor/I2P 站点检测。2026 年 4 月发布 v0.6.0，Star 数 21,500+，是 sherlock 项目的高性能演进版。适用于安全研究、数字足迹分析和调查取证。

### 🔥 Warp：AI 原生的 Agentic 开发终端

**来源**: GitHub | [项目链接](https://github.com/warpdotdev/warp)

Rust 编写的下一代终端模拟器，2026 年 4 月 28 日正式开源（AGPL v3），由 OpenAI 担任创始赞助商。Warp 不仅是高性能终端，更是一个"Agentic Development Environment"——内置 Oz 智能体平台，支持自然语言驱动的代码编写、调试和部署。单日 GitHub 增长超 12,800 星，已服务 70 万+ 开发者。其核心创新在于"开源 + Agent 驱动"的开发模式：AI Agent 负责代码实现，人类提供方向和决策。

### 🔥 mattpocock/skills：TypeScript 大神的 .claude 配置库

**来源**: GitHub | [项目链接](https://github.com/mattpocock/skills)

知名 TypeScript YouTuber Matt Pocock 将自己的 .claude 目录开源，一夜登顶 GitHub 全球 Trending 第一，24 小时内收获 22,000 星，目前 Star 数突破 45,000。项目包含一系列可复用的 AI 技能指令（SKILL.md 文件），覆盖 TypeScript 严格模式、联合类型模式、Zod schema 约定、类型安全 API 客户端生成等生产级模式。每个 skill 都经过实战检验，可直接用于 Claude Code 工作流，是 AI 辅助编程领域最热门的"技能包"项目之一。

### 🔥 TanStack AI：框架无关的 AI 工具链 SDK

**来源**: GitHub | [项目链接](https://github.com/TanStack/ai)

🤖 TanStack 团队推出的 AI SDK，秉承其标志性的 headless、类型安全理念。提供统一的 Provider 接口，支持 OpenAI、Anthropic、Google Gemini 等多模型切换而无需改动业务代码。核心特性：多模态内容支持（文本/图像/音频/视频/文档）、图像生成（DALL-E/GPT-Image/Imagen）、流式聊天与 Agent 循环策略、同构类型安全工具（服务端/客户端执行）、与 TanStack Start 深度集成、结构化可观测性事件。2026 年 4 月新增 ElevenLabs 语音/音频适配器，支持音乐合成和音效生成。

### 🔥 DeepSeek-V4：百万 Token 上下文开源大模型

**来源**: GitHub | [项目链接](https://huggingface.co/deepseek-ai)

2026 年 4 月 24 日发布的 DeepSeek V4 预览版，采用 MoE（混合专家）架构，提供两个版本：V4-Pro（1.6 万亿总参数 / 49B 激活）和 V4-Flash（2840 亿总参数 / 13B 激活），均支持原生百万 Token 上下文窗口。Apache 2.0 开源许可，同步发布技术报告和 API。在长上下文和 Agent 能力上达到与顶级闭源模型相当的水平，API 价格仅为 GPT-5.5 的 1/70。Hugging Face Transformers 已于 5 月 2 日完成集成。
