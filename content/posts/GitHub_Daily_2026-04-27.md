# GitHub 热门项目日报 · 2026-04-27

> 每天 5 分钟，了解 GitHub 上最值得关注的开源项目。

---

### 🔥 ml-intern

**来源**: GitHub | [项目链接](https://github.com/huggingface/ml-intern)

Hugging Face 团队推出的**开源 ML 工程师**，一个基于 smolagents 框架构建的自主 AI Agent，专门用于自动化机器学习后训练全流程。它能自主完成文献综述、数据集发现、训练脚本执行、失败诊断等迭代研究环节，Star 数已突破 2200+。适合需要快速实验和迭代 ML 模型的团队和个人研究者。

---

### 🔥 Hermes Agent

**来源**: GitHub | [项目链接](https://github.com/NousResearch/hermes-agent)

由 Nous Research 开发的**自我进化 AI Agent 框架**（Star 7.5万+），安装后连接你的消息账户，即成为持久化的个人助理——学习你的项目、构建自己的技能、定时运行任务。最新 v0.9.0 版本带来多智能体架构升级、可插拔内存提供程序、CamoFox 反检测浏览器等特性。支持任意模型，通过 Nous Portal 订阅实现零配置接入。

---

### 🔥 FinceptTerminal

**来源**: GitHub | [项目链接](https://github.com/Fincept-Corporation/FinceptTerminal)

现代化金融应用，提供高级市场分析、投资研究和经济数据工具。面向交互式探索和数据驱动决策而设计，适合金融从业者和量化研究者。支持股票、量化交易、财务分析等场景，是金融科技领域的开源 Bloomberg Terminal 替代方案。

---

### 🔥 browser-use

**来源**: GitHub | [项目链接](https://github.com/browser-use/browser-use)

让网站能被 AI Agent 访问和操作的开源框架。通过标准化接口，AI Agent 可以像人类一样浏览网页、填写表单、点击按钮。目前在 Python 生态中增长迅猛，是 AI Agent 落地 web 自动化场景的核心基础设施。

---

### 🔥 CUA (Computer-Using Agent)

**来源**: GitHub | [项目链接](https://github.com/trycua/cua)

由 Trycua 推出的**计算机使用智能体开源基础设施**（Star 13.5k+），支持 macOS/Linux/Windows 跨平台沙箱运行。为 AI Agent 提供可控的操作系统级别操作能力，是当前 Computer Use 方向最受关注的开源实现之一。

---

### 🔥 activepieces

**来源**: GitHub | [项目链接](https://github.com/activepieces/activepieces)

开源工作流自动化平台，特性之一是支持约 **400 个 MCP 服务器**集成。提供可插拔的 AI Agent 工作流设计器，让非技术用户也能快速搭建自动化流程。适合企业级工作流编排和 AI Agent 协作场景。
