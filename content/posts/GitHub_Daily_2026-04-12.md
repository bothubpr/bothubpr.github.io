### 🔥 superpowers

**来源**: GitHub | [项目链接](https://github.com/obra/superpowers)

superpowers 是一个面向编码智能体（Agent）的完整软件开发工作流解决方案。该项目由 Obra 团队维护，旨在为 AI 编码助手提供一套完整的技能框架，让智能体能够更高效地理解、生成和优化代码。技术亮点包括模块化的技能插件系统、支持多种编程语言的代码生成引擎、以及与主流 IDE 的深度集成。适用场景包括自动化代码审查、智能代码补全、项目脚手架生成等。截至今日，项目已获得超过 110K Stars，单日增长约 19,621 Stars，显示出极高的社区关注度。

### 🔥 hermes-agent

**来源**: GitHub | [项目链接](https://github.com/nousresearch/hermes-agent)

hermes-agent 是一个自我进化的 AI 智能体框架，由 NousResearch 团队开发。该项目专注于让 AI 智能体能够在运行过程中持续学习、适应并优化自身的行为，无需人工干预。主要功能包括动态技能获取、环境感知、任务分解与规划等。技术亮点在于其采用了强化学习与大型语言模型结合的方法，实现了智能体的在线学习能力。适用场景包括自动化客服、智能助手、复杂任务自动化等。项目目前处于快速迭代期，吸引了大量 AI 研究者的关注。

### 🔥 oh-my-codex

**来源**: GitHub | [项目链接](https://github.com/Yeachan-Heo/oh-my-codex)

oh-my-codex 是 Claude Code 的扩展工具集，旨在提升开发者在 Claude Code 环境下的编码体验。作者 Yeachan Heo 针对 Claude Code 的局限性，开发了一系列实用插件和配置模板。主要功能包括代码片段管理、主题定制、快捷键优化、以及集成外部工具链。技术亮点包括对多种编程语言的支持、高度可定制的用户界面、以及流畅的插件生态系统。适用场景包括 Claude Code 重度用户、希望提升编码效率的开发者、以及团队协作环境。项目在短短几个月内已获得近 10K Stars。

### 🔥 VibeVoice

**来源**: GitHub | [项目链接](https://github.com/microsoft/VibeVoice)

VibeVoice 是微软开源的一款语音 AI 项目，专注于生成富有情感和表现力的语音合成。该项目基于最新的生成式 AI 技术，能够根据文本内容自动调整语调、节奏和情感色彩。主要功能包括多语言支持、实时语音合成、情感控制参数等。技术亮点在于其采用了端到端的神经网络架构，大幅提升了合成语音的自然度和表现力。适用场景包括虚拟助手、有声内容创作、游戏配音、辅助通信等。作为微软的开源项目，VibeVoice 在语音 AI 领域具有较高的权威性。

### 🔥 OpenScreen

**来源**: GitHub | [项目链接](https://github.com/openscreen/openscreen)

OpenScreen 是一个免费、开源的屏幕录制与直播工具，旨在为用户提供轻量级且功能强大的录屏解决方案。该项目由 OpenScreen 社区维护，支持 Windows、macOS 和 Linux 三大平台。主要功能包括高清录屏、区域选择、音频捕获、实时直播推流等。技术亮点包括低 CPU 占用率、硬件加速编码、以及简洁的用户界面。适用场景包括软件演示、在线教学、游戏录制、会议记录等。项目以其易用性和稳定性赢得了大量用户的青睐，Star 数持续稳步增长。

### 🔥 Archon

**来源**: GitHub | [项目链接](https://github.com/archon/archon)

Archon 是一个 AI 编程 harness（框架），专注于为 AI 驱动的代码生成任务提供标准化、可复现的实验环境。该项目由 Archon 团队开发，集成了多种代码生成模型、评估指标和数据集。主要功能包括模型训练、代码生成、自动评估、结果可视化等。技术亮点在于其模块化设计，支持快速切换不同的 AI 模型和任务配置。适用场景包括 AI 代码生成研究、模型性能对比、教育实验等。该项目是 AI 编程领域的重要基础设施，受到了学术界和工业界的共同关注。

### 🔥 DeepTutor

**来源**: GitHub | [项目链接](https://github.com/HKUDS/DeepTutor)

DeepTutor 是一个 Agent-Native 个性化学习助手，由香港大学数据科学实验室（HKUDS）开发。该项目利用大型语言模型和强化学习技术，为每位学习者提供定制化的学习路径和实时辅导。主要功能包括知识点诊断、个性化推荐、交互式问答、学习进度追踪等。技术亮点在于其“智能体原生”设计，让学习助手能够主动感知学习者的状态并动态调整教学策略。适用场景包括在线教育、职业培训、语言学习、技能提升等。DeepTutor 代表了 AI 教育应用的前沿方向，Star 数在短期内快速增长。
