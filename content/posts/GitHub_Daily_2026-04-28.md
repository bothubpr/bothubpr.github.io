# GitHub 热门项目日报 · 2026-04-28

---

### 🤗 ml-intern

**来源**: GitHub | [项目链接](https://github.com/huggingface/ml-intern)

由 Hugging Face 团队开源的自动化机器学习工程助手，能够阅读论文、训练模型并部署 ML 模型。该项目将整个 ML 工作流自动化——从文献调研到模型打包上线，让研究人员专注于创新而非重复性工程工作。适合正在研究前沿模型、希望快速验证想法的 ML 团队。

### 🔧 free-claude-code

**来源**: GitHub | [项目链接](https://github.com/Alishahryar1/free-claude-code)

一款让用户在终端、VSCode 插件或 Discord 中免费使用 Claude Code 的开源工具，支持多种接入方式。项目持续活跃更新（最近更新：2026-04-26），为开发者提供了灵活的消费级 AI 编程助手替代方案。适用于预算有限但希望体验 Claude Code 能力的个人开发者。

### 🛠️ mattpocock/skills

**来源**: GitHub | [项目链接](https://github.com/mattpocock/skills)

前 Vercel 工程师 Matt Pocock 精选的 Claude Code Skills 集合，直接从他的 .claude 目录中提炼而出。覆盖 TypeScript、Shell 等多种语言和工具的场景化提示词，是学习和借鉴 AI Agent 高效提示词工程的优质资源库。

---

### 🧠 GitNexus

**来源**: GitHub | [项目链接](https://github.com/abhigyanpatwari/GitNexus)

一款零服务器的代码智能引擎，完全在浏览器端运行。用户只需上传 GitHub 仓库或 ZIP 文件，即可获得带有内置 Graph RAG Agent 的交互式知识图谱，大幅提升代码探索和理解效率。适用于代码审查、知识传承和大型代码库的快速熟悉场景。

---

### 📈 FinceptTerminal

**来源**: GitHub | [项目链接](https://github.com/Fincept-Corporation/FinceptTerminal)

现代化的金融分析应用，提供高级市场分析、投资研究和经济数据工具，交互体验友好，面向数据驱动的投资决策场景。被称为"开源版 Bloomberg Terminal"，支持股票、量化金融等多维度数据探索，适合散户投资者和金融科技开发者参考。

---

### ⚡ thunderbolt

**来源**: GitHub | [项目链接](https://github.com/thunderbird/thunderbolt)

由 Thunderbird 团队推出的本地 AI 助手框架，主打"选择你的模型，拥有你的数据"，帮助用户在本地运行大模型，摆脱云端厂商绑定。支持 on-device AI 和多种 LLM 接入方式，关注隐私和可控性的用户值得关注。
