### 🔥 Hermes Agent

**来源**: GitHub | [项目链接](https://github.com/NousResearch/hermes-agent)

Hermes Agent 是由 Nous Research 开发的自我改进型 AI Agent，拥有内置的学习闭环系统。它能够从经验中创建技能，在使用过程中不断改进，搜索自己的过往对话，并建立对用户需求的深度理解。今日新增 5,884 stars，总星数已达 45,575。

- **项目作者/组织**: Nous Research
- **主要功能/用途**: 提供可成长的 AI Agent 平台，支持技能创建、记忆管理和多平台集成
- **技术亮点/创新点**: 内置学习循环、持久化知识、自适应技能提升
- **适用场景**: 个人 AI 助手、自动化任务处理、代码辅助、多轮对话系统
- **Star 趋势**: 今日新增 5,884 stars，总星数 45,575（Python）

### 🔥 Superpowers

**来源**: GitHub | [项目链接](https://github.com/obra/superpowers)

Superpowers 是 Andrej Karpathy 推出的 Agentic 技能框架，使用 Shell 脚本实现模块化的 AI 能力组合。该框架强调轻量、可组合的技能单元，让开发者能够快速构建和部署 AI Agent。今日新增 2,299 stars，总星数突破 143,974。

- **项目作者/组织**: obra（Andrej Karpathy）
- **主要功能/用途**: 提供一套 Shell 脚本库，用于定义、调用和管理 AI Agent 技能
- **技术亮点/创新点**: 极简设计、技能可组合性、无缝集成现有工具链
- **适用场景**: AI 工作流自动化、快速原型验证、教育演示
- **Star 趋势**: 今日新增 2,299 stars，总星数 143,974（Shell）

### 🔥 MemPalace

**来源**: GitHub | [项目链接](https://github.com/MemPalace/MemPalace)

MemPalace 是由 Milla Jovovich 开发的 AI 记忆框架，采用“记忆宫殿”技术，在 LongMemEval 基准测试中获得 96.6% 的高分。该项目专注于为 AI Agent 提供长期记忆能力。今日总星数 35,693。

- **项目作者/组织**: Milla Jovovich
- **主要功能/用途**: 为 AI Agent 提供结构化记忆存储与检索，支持记忆关联、时间线回溯
- **技术亮点/创新点**: 记忆宫殿算法、高精度记忆召回、可扩展的记忆后端
- **适用场景**: 长期对话系统、个性化推荐、历史记录分析
- **Star 趋势**: 总星数 35,693（Python）
