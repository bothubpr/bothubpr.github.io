# GitHub 热门项目日报 · 2026-04-26

> 每天 5 分钟，了解 GitHub 上最值得关注的开源项目。

---

### 🤖 ml-intern

**来源**: GitHub | [项目链接](https://github.com/huggingface/ml-intern)

Hugging Face 官方推出的开源 ML 实习生 agent，能够阅读论文、训练模型并上线 ML 模型。项目刚开源即斩获 3800+ stars，增长迅猛。它打通了 LLM 后训练（post-training）的全流程——从文献调研到模型微调再到部署，统统自动化。对于需要快速实验不同训练策略的团队来说，这个工具能大幅提升迭代效率。适用场景：ML 研究团队、AI 初创公司模型开发流程加速。

---

### 🧠 Hermes Agent

**来源**: GitHub | [项目链接](https://github.com/NousResearch/hermes-agent)

NousResearch 出品的"与你共同成长的 AI 智能体"，目前累计 47,577 stars，4 月单月新增超 6000+ stars，已成长为最炙手可热的 AI Agent 框架之一。核心亮点在于**自我进化**能力——随着使用不断学习你的偏好和项目上下文，愈发懂你。v0.9.0 版本支持后台任务自动通知、实时模型切换、Chrome CDP 浏览器连接等。配套的 Hermes Agent Self-Evolution 项目还引入了 DSPy + GEPA 自动优化 prompt 和技能描述。安装仅需一行 shell 命令，对个人助手和团队自动化场景都非常友好。

### 📱 Google AI Edge Gallery

**来源**: GitHub | [项目链接](https://github.com/google-ai-edge/gallery)

Google 官方推出的端侧 AI 应用Gallery，支持在手机上本地运行 Gemma、LLaMA 等大语言模型，完全离线、隐私优先。当前一周新增 4326 stars，总量已达 20,100+。项目同时提供 Android APK 和模型转换工具链，开发者可以在 repo 内找到详尽的 Wiki 指南。适用场景：移动端 AI 应用开发、端侧大模型部署实验、设备端隐私推理。

---

### 🖥️ Open WebUI

**来源**: GitHub | [项目链接](https://github.com/open-webui/open-webui)

热门的自托管 AI 聊天界面，累计 127K stars，支持 Ollama、OpenAI API 等多种后端，内置 RAG、MCP 扩展等企业级功能。305,000+ 用户已部署使用，活跃度极高。近期版本强化了多模型管理和 WebSearch 插件。对希望私有化部署 AI 对话能力的团队和个人开发者来说，是目前最成熟的开源解决方案之一。

---

### 🔄 active-project

**来源**: GitHub | [项目链接](https://github.com/active-project/active-project)

一款将 AI 趋势同步到代码库的工具，自动发现 GitHub trending 仓库并匹配到你的项目代码中，支持 MCP、多 Agent 协作和 RAG。针对 AI coding agent 工具链的不断演进，这个项目可以帮助团队保持工具配置的与时俱进。适合：AI 工具链维护者、关注最新 AI coding 实践的开发者。

---

### 🛡️ KeygraphHQ / Shannon

**来源**: GitHub | [项目链接](https://github.com/KeygraphHQ/shannon)

Shannon Lite 是一款白盒 AI 渗透测试工具，能够分析 Web 应用和 API 的源代码，识别攻击向量并执行真实漏洞利用，在生产环境前主动证明漏洞存在。面向安全工程师和 DevSecOps 团队，提供自动化渗透测试能力，帮助在 CI/CD 流程中内嵌安全检测。
