Axios 被植入恶意版本，NPM 供应链遭攻击并释放远控木马#

来源： Hacker News

知名 HTTP 客户端库 Axios 在 NPM 上被攻击者入侵，多个恶意版本被发布，意图向开发者机器植入远程访问木马（RAT）。安全公司 StepSecurity 披露了完整攻击链，建议开发者立即检查并降级至安全版本。

📎 链接：StepSecurity 完整报告

Claude Code 源代码通过 NPM registry 中的 map 文件遭泄露#

来源： Hacker News

Anthropic 旗下 AI 编程工具 Claude Code 的源代码被人通过 NPM registry 中的 map 文件泄露，在社交媒体引发广泛讨论，相关话题在 HN 上获得超 1000 点热度。

📎 链接：Twitter 讨论帖

GitHub 迫于压力撤回 Copilot pull request 广告功能#

来源： Hacker News

GitHub 在强烈反对声中宣布取消在 pull request 中插入 Copilot 广告的功能，此前开发者普遍认为该行为损害了开源生态，The Register 报道了这一争议事件。

📎 链接：The Register 报道

Ollama 预览版已支持 Apple Silicon 的 MLX 框架#

来源： Hacker News

本地大模型运行工具 Ollama 发布博客，宣布其预览版现已支持 Apple Silicon 上的 MLX 框架（由 MLX 团队推出），显著提升在 Mac 设备上的 AI 推理效率。

📎 链接：Ollama 官方博客

Google 发布 2 亿参数时间序列基础模型 TimesFM，支持 16k context#

来源： Hacker News

Google Research 开源了 TimesFM，一个拥有 2 亿参数的时间序列预测基础模型，支持最高 16k token 的上下文长度，在时间序列预测任务上展现出强大泛化能力。

📎 链接：TimesFM GitHub

Artemis II 任务被指存在安全隐患，NASA 登月计划蒙阴影#

来源： Hacker News

技术博主对 NASA Artemis II 任务的安全性提出严重质疑，指出多项关键技术问题尚未解决，可能导致载人登月任务延期或取消，引发航天界广泛讨论。

📎 链接：Idle Words 原文分析

Android Developer Verification 向所有开发者全面推开#

来源： Hacker News

Google 宣布 Android Developer Verification 功能向所有开发者开放，开发者需通过身份验证才能在 Play Store 发布应用，旨在提升应用生态安全性，减少恶意软件传播。

📎 链接：Android Developers 官方博客

开源 CAD 软件 Solvespace 推出浏览器版本#

来源： Hacker News

开源参数化 CAD 工具 Solvespace 推出纯浏览器版本，无需安装任何软件即可在 Web 端进行 3D 建模，标志着开源 CAD 向 Web 平台的重要迈进。

📎 链接：Solvespace Web 版

Chrome 将 yt-dlp 标记为可疑下载，引发开源工具争议#

来源： Hacker News

Chrome 浏览器在用户尝试下载最新 yt-dlp（热门视频下载工具）时弹出「可疑下载」警告，但未提供任何解释，引发开发者对浏览器安全策略的质疑。

📎 链接：Hacker News 讨论

Microsoft 在 Copilot 个人版服务条款中新增措辞，明确其 AI 助手「仅供娱乐目的使用」，引发用户对 Copilot 专业应用可靠性的广泛讨论。

📎 链接：Microsoft Copilot 服务条款
